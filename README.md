# EVE-online-analytic

Набор данных и результаты анализа Monthly Economic Report (MER) по EVE Online.

## Структура проекта
- `Документы/` — пояснительные записки и документация.
- `данные/исходные/архивы/` — части исходного архива (Git LFS).
- `данные/обработанные/сводные_таблицы/` — агрегированные CSV‑таблицы (Git LFS).
- `Результаты анализа/` — графики и отчеты по анализу.
- `скрипты/` — скрипты обработки и анализа.

## Восстановление архива из частей (Windows)
Перейдите в `данные/исходные/архивы/` и выполните:

```bat
copy /b архивы.zip.part001+архивы.zip.part002+архивы.zip.part003 архивы.zip
```

После восстановления можно распаковать `архивы.zip` любым архиватором.

## Примечания
- Крупные файлы хранятся через Git LFS.
- Распакованные данные не хранятся в репозитории.
- Файл пояснительной записки в PDF лежит в корне проекта.
